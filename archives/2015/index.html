  6. PocketSDK-iOS版本更新日志

    v3.2.8

    废除继续支付功能，响应地：

    1. 正式账号支付成功后的提示弹窗，取消继续支付按钮，保留返回游戏按钮。

    2. 新接口（使用方法请参见接入文档）：

       + (void)startPayWithGameOrderId:(NSString *)gameOrderId roleInfo:(PocketSDKRoleInfo *)roleInfo;
 + (void)buyProductWithIdentifier:(NSString *)productId gameOrderId:(NSString *)gameOrderId roleInfo:(PocketSDKRoleInfo *)roleInfo;
 + (void)didEnterGameWithRoleInfo:(PocketSDKRoleInfo *)roleInfo;

    3. 新接入游戏不需要实现以下两个回调方法：

       - (NSString *)gameOrderId;
 - (NSDictionary *)roleInfoInTheGame;

    4. 新接入游戏不再使用以下方法：

       + (void)showPaymentView;
 + (void)buyProductWithIdentifier:(NSString *)productId;
 + (void)roleDidEnterGame;

    v3.2.7

    • 新增官方支付接口+ (void)buyProductWithIdentifier:(NSString *)productId;以及相应回调方法- (void)inAppPurchaseDidSuccess;和 - (void)inAppPurchaseDidFailWithReason:(PocketSDKIAPFailReason)reason;

    V3.2.6

    • 解决了在unity3D游戏中，游戏从后台恢复悬浮按钮被覆盖的问题。

    V3.2.5

    • 解决了cocos2dx触摸穿透的问题。

    2015-01-07 • iOS
