抽象层日志系统接入指南 V1.0

一、概述

本文档用于指导WEB客户端快速接入PocketGames抽象层日志系统服务器端

二、代码接入详细介绍

1.初始化日志接口

  • 请求URL地址

    http://ip地址:端口/pocketgames/client/log/init

  • 请求参数

    HTTP协议，支持POST方法

    参数名称 类型 示例值 必须 描述
    appId int 是 平台方分配给游戏的appId
    source int 是 平台来源0=ios 1=android 2=web
    packageId int 是 包ID
    network int 是 网络 0=wifi 1 = 3g 2=其他
    model String 是 机型
    operatorOs String 是 操作系统
    deviceNo String 是 设备号(ANDROID=IMEI,IOS=IDFV)
    device String 是 device(Android=MAC#ANDRIDID IOS=IDFA）
    version String 是 游戏版本
    sdkVersion String 是 SDK版本
    clientTime String 是 客户端时间 (yyyy-MM-dd HH:mm:ss)
    sign String 是 MD5(appId+source+packageId+appKey)

  • 返回参数

    以JOSN格式返回结果

    称 类型 示例值 描述
    code int 返回状态 200=成功,其他等于失败
    error_msg String 错误信息

    sdkVersion String 是 SDK版本
    clientTime String 是 客户端时间 (yyyy-MM-dd HH:mm:ss)
