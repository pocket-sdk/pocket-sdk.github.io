记录Swift的各种小知识。


Guard语句

func someFunc(x: Int?) 
{
    guard let x = x where x > 0 else {
        return;
    }
    // 如果没有进入else，guard let x = x 已经对 x: Int? 进行了一次解包。   
    var sum = 3 + x
}

保证 x不为nil并且大于0 否则 return

等价于

func someFunc(x: Int?)
{
    if x = nil || x <= 0 {
        return;
    }
    // ...
}

不同之处在于，使用guard我们只需要考虑符合我们要求的值是什么，直接将不符合的值过滤掉（如someFunc中的return）。
