在Mac上搭建PHP开发环境

1. Apache

下载安装

Mac自带Apache。

有两种方法启动：

  1. 打开”系统偏好设置”->”共享”，在”互联网共享”那一项前面打√。（未测试）

  2. 终端启动并查看版本信息。

    sudo apachectl start        
sudo apachectl －v

配置

  1. 配置socket

  2. 配置缺省路径

    sudo apachectl start

sudo apachectl －v

2. MySQL

下载安装

  1. 前往MySQL官方网站下载并安装。

  2. 在”系统偏好设置”->”MySQL”中启动MySQl。

  3. 终端设置root账户密码：

    /usr/local/mysql/bin/mysqladmin -u root password 12345

  4. 登录数据库

    mysql -u root -p

配置

  • 设置数据库的编码格式为UTF8

    1. 拷贝/etc/my-default.cnf至/etc/my.cnf或其他位序的目录。（mysql –help查看Default options are read from the following files in the given order:…）

    2. 在[client]部分加入：default-character-set=utf8

      在[mysqld]部分加入：character-set-server=utf8
