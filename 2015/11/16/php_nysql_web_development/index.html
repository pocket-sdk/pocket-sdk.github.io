  • 可变变量：允许动态地改变一个变量的名称

    $varName = 'age';
$$varName = 18;
// 等价于$age = 18;

  • 变量的作用域

    • 内置超级全局变量：任何地方都可见
    • 常量：全局可见

    • 全局变量：脚本可见，函数内部不可见

      $globalVar;
function someoneFunc() {
    // 想要在函数内部使用全局变量，需要在函数内声明该全局变量。
    global $globalVar;
}

    • 函数内变量：函数内（函数内的静态变量）

  • 引用操作符&：引用像是一个别名，而非指针（类似于C++）。

    $a = 5;    $b = &$a;
unset($a); // 取消$a和5的链接，isset($a)为false

fucntion someone_fincton(&$param) {
    // 参数$param是其自身，而不仅仅是它的值
}

  • 执行操作符`：当做服务端命令来执行。

  • switch的条件值可以是整型、字符串或浮点型

  • 数组排序

    升序排序：sort($array[, SORT_REGULAR])、asort()（按值排序）、ksort()（按key排序）

    反向排序：rsort()、arsort()、krsort()

    用户定义排序：usort($array, ‘method_name’)

  • 将数组元素随机：shuffle()

    将数组元素反向：array_reverse()
